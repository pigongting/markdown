# 静态路由

以前版本的 React Router 使用静态路由来配置应用程序的路由。这允许在渲染之前对路线进行检查和匹配。由于 v4 移动到动态组件而不是路由配置，一些以前的用例变得不那么明显和棘手。

我们正在开发一个使用静态路由配置和 React Router 的软件包，以继续满足这些用例。它正在开发中，但我们很乐意为您尝试并帮助您。
